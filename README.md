# My dotfiles

## How to run

```bash
$ sh install
```

## Instalation

```bash
$ curl -L https://raw.github.com/gbrlsepulveda/dotfiles/master/install | sh
# or
$ git clone https://github.com/gbrlsepulveda/dotfiles.git && cd dotfiles && sh install.sh
```
